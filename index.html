Problema Solución Misión Valores Descargar

Problema

Las dificultades para comprender conceptos técnicos de hardware y software. La falta de fuentes accesibles, claras y adaptativas de información tecnológica.

Solución

Una aplicación basada en inteligencia artificial que responde preguntas, dando una interfaz intuitiva con respuestas para el usuario y con recursos visuales interactivos para mejorar la comprensión.

Misión

Abrir el acceso al conocimiento tecnológico, facilitando la comprensión de conceptos complejos de hardware y software a través de una aplicación educativa, fácil de usar.

Visión

Expandir nuestros horizontes llevando este proyecto a toda persona posible. Queremos ser el líder en educación tecnológica, alcanzando reconocimiento global como una aplicación que transformó la manera de aprender.

Valores Fundamentales

  • Accesibilidad
  • Innovación continua
  • Educación abierta
  • Integridad y veracidad
  • Empoderamiento
  • Inclusión digital
  • Orientación al usuario
